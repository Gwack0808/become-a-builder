Concepts I've learned by building things

📄

HTML

The skeleton of a webpage. Tags describe what things are — a heading, a card, a button.

Fun fact

HTML was invented by Tim Berners-Lee in 1991. The first webpage is still live at info.cern.ch.

🎨

CSS

The styling layer. CSS controls how things look — colors, fonts, spacing, and layout.

Fun fact

CSS was proposed in 1994. Before it existed, styling was done with HTML attributes directly — one reason old websites look so chaotic.

🚢

Shipping

Making something real and putting it somewhere people can see it. The opposite of keeping it on your computer forever.

Fun fact

"Ship it" comes from the physical act of shipping software on disks. Now it just means: make it live.

🌐

GitHub Pages

A free service that takes files in a GitHub repo and serves them as a real website. No server required.

Fun fact

GitHub Pages has hosted millions of free sites since 2008 — portfolios, docs, games, and tools built by people just like you.

📸

Git

Version control. Every commit is a saved snapshot of your code. Push sends it to GitHub — and live to the world.

Fun fact

Git was created by Linus Torvalds in 2005 — the same person who created the Linux operating system.

⚡

JavaScript

The behavior layer. JS makes pages respond to the user — clicks, inputs, animations, and live updates.

Fun fact

JavaScript was written in just 10 days in 1995. Despite the name, it has nothing to do with Java.
